24/7 ⏱ Uptime Python Script

Keep Your Server
Online 24/7

One command to keep your free VPS or CodeSandbox devbox awake and your server running around the clock. Works with Minecraft servers, Pterodactyl panels, PufferPanel, and more.

01 Works With Compatible Setups
⛏MC-CODESANDBOX
🦅PTERODACTYL-1
🦅PTERODACTYL-2
🦅PTERODACTYL-CB
🐡PUFFERPANEL
ℹ Run this after your server or panel is already started. It keeps the environment alive so it doesn't go to sleep. Follow the video to see exactly when to run it in your setup.
02 Run the Script One Command

Start the 24/7 uptime script

⚠ Keep this running in its own terminal tab. Closing the terminal or stopping this script will allow the environment to sleep and your server to go offline.
bash
python3 <(curl -s https://raw.githubusercontent.com/unnamed-boy07/pterodactyl/refs/heads/main/24-7)
