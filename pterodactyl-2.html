PTERODACTYL-2 ⚙ Pterodactyl Script Install

Install Pterodactyl Panel +
Wings via Script

One-command script-based installation of the Pterodactyl panel and Wings daemon. Faster than the manual Docker method — ideal for fresh VPS setups. Includes game server IP setup via playit.gg and ngrok.

01 · Panel Setup 02 · Wings Setup 03 · Game Server IP
01 Panel Setup Auto Script

Run the panel install script

ℹ Follow along with the video as the script will prompt you for database passwords, your domain/IP, and mail settings during setup.
bash
bash <(curl -s https://raw.githubusercontent.com/unnamed-boy07/pterodactyl/refs/heads/main/pt-panel)
02 Wings Setup Auto Script + Config

Run the Wings install script

bash
bash <(curl -s https://raw.githubusercontent.com/unnamed-boy07/pterodactyl/refs/heads/main/pt-wings)

Configure Wings — follow these steps in order

⚠ Steps 4 & 5 require you to copy the config token from your panel's node page — watch the video to see exactly where to find it.
1 Do as shown in the video (complete the script prompts)
2 Navigate to the pterodactyl directory: cd pterodactyl
3 Switch to root: sudo su
4 Open the Wings config file: nano /etc/pterodactyl/config.yml
5 Paste the config copied from your panel node page (as shown in the video)
6 Go into Wings folder: cd wings
7 Restart Wings to apply the config: docker-compose up -d --force-recreate

Step 2–3 — Navigate & root

bash
cd pterodactyl sudo su

Step 4 — Open Wings config

bash
nano /etc/pterodactyl/config.yml

Step 6–7 — Restart Wings

bash
cd wings docker-compose up -d --force-recreate
03 Creating a Game Server IP playit.gg / ngrok

Option A — playit.gg (recommended for game servers)

ℹ playit.gg gives your game server a stable public IP and port without any firewall changes. Best option for Minecraft and other game servers.
bash
wget https://github.com/playit-cloud/playit-agent/releases/download/v0.15.26/playit-linux-amd64 chmod +x playit-linux-amd64 ./playit-linux-amd64

Option B — ngrok (for panel HTTP tunnel)

⚠ Install ngrok first using the block below, then run the tunnel command. You need an ngrok account and auth token for persistent tunnels.
bash — install ngrok
curl -sSL https://ngrok-agent.s3.amazonaws.com/ngrok.asc \ | tee /etc/apt/trusted.gpg.d/ngrok.asc >/dev/null \ && echo "deb https://ngrok-agent.s3.amazonaws.com bookworm main" \ | tee /etc/apt/sources.list.d/ngrok.list \ && apt update \ && apt install ngrok

Start the ngrok tunnel

bash
ngrok http http://localhost:443
