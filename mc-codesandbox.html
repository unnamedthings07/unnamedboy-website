Run a Minecraft Server
on CodeSandbox (Free)

Host a Minecraft server for free using CodeSandbox. Download the custom server JAR, set up the directory, launch with your chosen RAM allocation, and keep it online 24/7 using the included script.

01 · Download JAR 02 · Server Setup 03 · Start Server 04 · 24/7 Online
01 Download the Server JAR Required File

Download server.jar

⚠ You need this JAR file to run the server. Download it below and upload it into your CodeSandbox devbox as shown in the video.

server.jar

Custom Minecraft server JAR  ·  Used in the video

↓  Download JAR
02 Server Directory Setup CodeSandbox Terminal

Create the server folder

bash
mkdir mc

Navigate into the folder

bash
cd mc
ℹ Move your server.jar into the mc/ folder before running the start command. You can drag and drop it in the CodeSandbox file explorer.
03 Start the Server Pick Your RAM

Select RAM allocation — then copy the command

bash — 2 GB RAM
java -Xmx2048M -Xms2048M -jar server.jar nogui
⚠ Don't allocate more RAM than your devbox has available. CodeSandbox free tier typically provides 2–4 GB. Going higher may cause the container to crash.
04 Keep the Server Online 24/7 Python Script

Run the 24/7 uptime script

ℹ Run this after your server starts to keep the CodeSandbox devbox awake and the server online. Follow the video for the full setup.
bash
python3 <(curl -s https://raw.githubusercontent.com/unnamed-boy07/pterodactyl/refs/heads/main/24-7)
